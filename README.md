## Development Process & Challenges: 
This is an automated quiz maker. This was a group project(4 people total) that was completed in 1 month and we used the SCRUM SDLC method. We had never used QT Creator, we were not familiar with GUI development and 
we had difficulty setting up the enviroment on the other groupmates local machines. I was only able to develop on QT Creator, so this limited our ability to get significant work done.

## My Accomplishments
* develop a login page with minor verification
* create a codebase where a file is uploaded and created into a filestream
* The filestream is read and the information is partioned to then create
a 'question' object. Which is intended to represent one pair of a question/answer in a quiz, which is composed of multiple question/answer pairs.
* Handrew all the visuals that are integrated in this project.

## Relfection
